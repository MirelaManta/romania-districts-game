## Romania Districts Game ##
#### Do you know all the counties in your country? ###  
__I have created a little exercise that tests your knowledge about Romania's counties.__ 
* The map, as well as the names of the counties at the right coordinates, were created using the turtle module.  
* The role of the pandas library was to read this data from a csv file, and to save in a new file all the counties that the player did not guess when the game ends.
